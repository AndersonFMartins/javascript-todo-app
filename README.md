# To-Do App

Aplicação de lista de tarefas desenvolvida com **HTML, CSS e JavaScript**.

## Funcionalidades

- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo

Projeto desenvolvido para praticar majoritariamente Javascript.
